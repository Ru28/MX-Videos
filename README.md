# MX-Videos


MX Videos is Video sharing platform. User can create account, watch video after login, Subscribe others, Like the videos, See Dashboard where users can analysis their videos. Developed a web application using with Django serving a REST API with templates.Implemented django OAuth model to get data from userâ€™s account.Users can update their Profiles.

## Set up

First, clone this repository:
```
git clone https://github.com/Ru28/MX-Videos.git
```

Install dependencies:
```
pip install -r requirements.txt
```

Apply migrations to your database
```
python manage.py migrate
```
To run the development server:
```
python manage.py runserver
```
